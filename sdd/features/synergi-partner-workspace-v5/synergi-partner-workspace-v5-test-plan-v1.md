# Synergi Partner Workspace v5 Test Plan v1

Fecha: 2026-03-23
Feature key: `synergi-partner-workspace-v5`

## Escenarios

1. Un referral con owner y stage se muestra tanto en el workspace como en la consola interna.
2. Un asset publicado puede aparecer como version actual y luego como retirado sin perder trazabilidad.
3. Una nueva version de un asset conserva la relacion con la version anterior.
4. Una solicitud de asset pack muestra delivery reference, method y notas de seguimiento.
5. El dashboard 360 refleja los nuevos indicadores sin romper el layout premium.
6. Las notificaciones destacan acciones reales y no solo estados genericos.
7. La experiencia se mantiene usable en resoluciones desktop y mobile.
